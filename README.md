# Roadway Website

This project is a **Roadway-inspired website developed using HTML, CSS, and JavaScript** as part of a college mini project. The website demonstrates a simple multi-page layout with navigation, product display, contact information, and a registration form.

## Project Structure

* **Home.html** : Main homepage of the website.
* **About.html** : Contains information about the company.
* **Products.html** : Displays product details.
* **Contact.html** : Provides contact information.
* **Registration.html** : Allows users to register through a form.
* **Validation.js** : JavaScript used for form validation.
* **images/** : Contains all images used in the website.

## Features

* Multi-page website design
* Product display section
* Contact and about pages
* User registration form
* Basic form validation using JavaScript
* Simple and clean user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **Home.html** in any web browser.

## Notes

This project was developed for learning basic **web development concepts** such as page navigation, form handling, and website structure.
